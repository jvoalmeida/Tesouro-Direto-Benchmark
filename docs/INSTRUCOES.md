# Como obter o extrato do Tesouro Direto

## Passo a passo

1. Acesse o Portal do Investidor do Tesouro Direto:  
   👉 **[https://portalinvestidor.tesourodireto.com.br](https://portalinvestidor.tesourodireto.com.br)**

2. Faça login com sua conta **Gov.br**

3. No menu lateral, clique em **Extrato**

4. Preencha os filtros:
   - **Instituição Financeira**: selecione sua corretora (ex: XP, Nubank, BTG, etc.)
   - **Ano**: selecione o ano desejado
   - **Mês**: selecione o mês desejado

5. Clique em **Aplicar** para carregar os dados

6. Na lista de títulos que aparecer, **clique no título** que deseja visualizar para expandir os detalhes

7. Clique no botão **Exportar** para baixar a planilha (arquivo `.xls`)

8. **Repita** os passos 4 a 7 para cada mês/ano que você quiser acompanhar

## Importando no Tesouro Direto Benchmark

Após baixar os arquivos:

1. Abra o **Tesouro Direto Benchmark** no seu navegador
2. Arraste os arquivos `.xls` para a área de importação **ou** clique nela para selecionar manualmente
3. Você pode importar vários arquivos de uma vez
4. O app detecta duplicatas automaticamente — se você reimportar o mesmo título, os dados serão atualizados

## Dicas

- **Importe múltiplos meses** para acompanhar a evolução dos seus títulos ao longo do tempo
- **Exporte em CSV** pelo botão no canto superior direito para analisar em planilhas (Excel, Google Sheets)
- Os dados ficam salvos **localmente no seu navegador** (localStorage). Nenhum dado é enviado para servidores externos
- Se quiser limpar todos os dados, use o botão **"Limpar tudo"** no cabeçalho do app
